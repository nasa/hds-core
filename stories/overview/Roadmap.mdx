## Planned for future releases

We're actively working to prioritize the post-v1.0 roadmap based on community needs.

**[Share what you need most](https://github.com/nasa/hds-core/discussions):**: upvote and comment on ideas in GitHub Discussions to help shape what we build next. Navigation components (Header, Footer, Secondary Nav) are our top priority for the first post-v1.0 release. Everything else will be driven by adopter demand.

## Additional components

All USWDS components work with HDS Core theming out of the box; colors, typography, and spacing apply automatically. Components listed above will receive dedicated HDS styling and documentation. USWDS components not listed here (button groups, collections, etc.) continue to work with the default USWDS appearance, themed to HDS.
