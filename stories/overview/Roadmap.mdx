## Planned for future releases

After v1.0, HDS Core will expand to cover more of the Horizon Design System. Planned additions include:

**Cards & layout** — Cards, image with caption, gallery thumbnails

**Interactive components** — Dropdown menus, modals, tooltips, tabs, date picker, search

**Status & feedback** — Alerts, status indicators, loaders, steppers

**Media** — Audio player, video player, carousels

**Extended form elements** — File upload, sliders, toggles, chips

**Data visualization** — Chart color palettes and accessibility guidance

These components require JavaScript and/or additional design work beyond the current CSS-only scope. Priorities will be based on community input — check the [GitHub Discussions](https://github.com/nasa/hds-core/discussions) to see what's being considered and to share your needs.

## Additional components

All USWDS components work with HDS Core theming out of the box — colors, typography, and spacing apply automatically. Components listed above will receive dedicated HDS styling and documentation. USWDS components not listed here (button groups, collections, etc.) continue to work with the default USWDS appearance, themed to HDS.
